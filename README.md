# Bone Mineral Density Prediction using Machine Learning

## Project Overview
This project predicts Bone Mineral Density (BMD) values using machine learning models.

## Dataset
Bone Mineral Density dataset containing:
- Age
- Gender
- Smoking Status
- Alcohol Consumption
- Medical attributes

Target:
BMD_Value

## Workflow
1 Data Cleaning
2 Data Exploration
3 Train-Test Split
4 Data Preprocessing
5 Model Building
6 Model Evaluation
7 Hyperparameter Tuning
8 Best Model Selection
9 Model Deployment

## Models Used
- KNN Regressor
- Decision Tree Regressor
- Linear Regression
- Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- Voting Regressor
- Stacking Regressor

## Evaluation Metrics
- R2 Score
- RMSE
- MSE

## Best Model
Linear Regression

## Deployment
Streamlit App deployed on HuggingFace Spaces.
